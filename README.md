# Ashutosh Sharma - Portfolio

This is the official professional portfolio website of **Ashutosh Sharma**, a Product Designer & UI/UX Specialist. It showcases selected works in mobile & web design, detailed case studies, skills analysis, and professional experience.

## Overview

Designed to be immersive and interactive, this portfolio bridges the gap between aesthetic excellence and functional efficiency. It features a modern, responsive interface with smooth animations, dark mode support, and a focus on user experience.

## Tech Stack

*   **Frontend Framework:** React 19
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **Animations:** Framer Motion
*   **Icons:** Lucide React
*   **Data Visualization:** Recharts

## Features

*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.
*   **Dark/Light Mode:** System-aware theme switching with persistence.
*   **Interactive UI:** Custom cursor logic, parallax effects, and micro-interactions.
*   **Project Showcase:** Filterable project grid distinguishing between Mobile and Web projects.
*   **Skills Visualization:** Radar chart implementation for skill analysis.
*   **Contact Form:** Integrated with Formspree for handling submissions.

## Sections

1.  **Hero:** Introduction with dynamic typography and parallax background.
2.  **About:** Professional summary.
3.  **Profiles:** Links to professional networks (Behance, LinkedIn, Dribbble, etc.).
4.  **Skills:** Visual representation of technical and design expertise.
5.  **Experience:** Career timeline and education history.
6.  **Projects:** Featured case studies (Saltoris, Penny Upp, Bondspe, Roofing, Chimney, etc.).
7.  **Design Showcase:** Interactive grid of visual work.
8.  **Testimonials:** Client and colleague feedback.
9.  **Contact:** Direct messaging interface.
